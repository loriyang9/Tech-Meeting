# 📥 網頁簡報版型使用指南 (Slide Template Starter Kit)

這是一個為您精心整理的「網頁簡報版型範本」，完整保留了「量產去塑研究啟始會議」的 **無印良品簡約風 (Paper & Zen Style)** 視覺設計與互動引擎。

本引擎最棒的特點在於 **「完全自動化」**：**不需修改任何 JavaScript 程式碼**！您只需要編輯 HTML 來新增、刪除投影片或側邊欄，系統就會自動計算頁數與底部進度條。

---

## 📂 檔案目錄結構

- 📄 `index.html` — **投影片內容與結構**（在這裡修改文字、新增投影片與側邊欄目錄）。
- 🎨 `index.css` — **視覺風格設計與配色**（在這裡修改品牌色、發光背景色、卡片樣式）。
- ⚡ `index.js` — **簡報控制引擎**（自動處理鍵盤方向鍵、滑鼠滾輪、平板手勢、進度條與全螢幕，無需修改）。

---

## 🛠 如何客製化您的新簡報？

### 第一步：修改簡報基本資訊
打開 `index.html`，在最前方的 `cover-slide` 部分修改您的簡報主標題、會議日期、專案階段以及核心團隊成員。

### 第二步：新增或刪除投影片（超簡單！）
本簡報採用語意化的 HTML5 標籤：

1. **修改側邊欄目錄**（`index.html` 約第 26 行）：
   每個側邊欄目錄項目都是一個 `<a>` 標籤，例如：
   ```html
   <a href="#slide-3" class="nav-item" data-slide="3">
       <span class="nav-num">02</span>
       <span class="nav-title">您的章節名稱</span>
   </a>
   ```
2. **修改投影片內容**（`index.html` 約第 72 行）：
   每一張投影片都是一個 `<article>` 標籤，您只需依序排列，並確保 `id` 對應側邊欄即可：
   ```html
   <article id="slide-3" class="slide">
       <div class="slide-content">
           <span class="slide-tag">02. CHAPTER SUBTITLE</span>
           <h2 class="slide-title">投影片主要標題</h2>
           
           <!-- 投影片內容：可放置文字、卡片或圖片 -->
           <p class="subtitle-desc">這裡是對應的描述文字...</p>
       </div>
   </article>
   ```

### 第三步：更換品牌配色（Theme Colors）
如果您想為不同的專案換一套品牌色（例如永續綠、防災紅、極簡灰），只需打開 `index.css`，在最上方的 `:root` 變數中直接修改色碼即可：
```css
:root {
    /* 背景與文字基本色 */
    --color-bg-primary: #FDFCF7;      /* 溫潤再生紙底色 */
    --color-text-primary: #2C2A29;    /* 深炭灰（比純黑更柔和） */
    
    /* 核心品牌色（在這裡直接修改，全站會同步變更！） */
    --color-accent-green: #3F5E4D;     /* 專案代表色（例如：生態松綠） */
    --color-accent-red: #8C2520;       /* 警示與強調色（例如：富迪防災紅） */
    --color-accent-sand: #C4A482;      /* 溫潤牛皮紙沙色 */
}
```

## 🖥 簡報播放快捷鍵

- **全螢幕播放**：按下鍵盤 **`F`** 鍵，即可進入/退出沉浸式全螢幕簡報模式。
- **切換投影片**：
  - 鍵盤：**左右方向鍵 `←` `→`**、**上下方向鍵 `↑` `↓`**、**空白鍵 `Space`** 或 **回車鍵 `Enter`**。
  - 滑鼠：直接滾動**滑鼠滾輪**即可順暢切換。
  - 平板與手機：支援**手勢左右滑動**切換。
- **快速跳轉**：點擊左側目錄即可直接跳轉至該章節。

## 🚀 發表與分享方式

1. **本機直接播放**：最簡單的方法，直接雙擊 `index.html` 用瀏覽器開啟，按下 `F` 鍵即可進行高質感發表！
2. **Google Drive 雲端分享**：您可以將整個資料夾同步至您的 Google Drive，方便隨時開啟與備份。
3. **網頁線上部署 (Vercel / GitHub Pages)**：如果您想給團隊或客戶一個專屬連結，只需將本資料夾上傳至 GitHub，並使用 Vercel 或 GitHub Pages 進行免費託管，即可擁有超流暢的線上簡報連結！
